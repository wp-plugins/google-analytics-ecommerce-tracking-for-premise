Send Ecommerce data from Premise to Google Analytics.

== Description ==

The Google Analytics Ecommerce Tracking for Premise allows you to send Ecommerce data from Premise to Google Analytics. It passes item and order information to Google Analytics reports when the Ecommerce option is activated for your profile.

In order to use this plugin you will need to have a copy of Premise.

Read more details about the plugin on the [Google Analytics Ecommerce Tracking for Premise](http://www.eugenoprea.com/wordpress-plugins/premise-google-analytics-ecommerce/) site.

== Installation ==

Please follow the [standard installation procedure for WordPress plugins](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins).

You can follow the manual procedure:

1. Upload the <code>premise-google-analytics-ecommerce</code> directory into the <code>/wp-content/plugins/</code> directory.
2. Activate the Google Analytics Ecommerce Tracking for Premise from the Plugins" section in your WordPress dashboard.
3. Add your Google Analytics profile id and activate the integration from Member Access > Google Analytics Ecommerce.
